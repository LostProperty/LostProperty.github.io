More about this project

Seabeard is a mobile game in which you follow in the footsteps of the great captain Seabeard and discover a giant ocean teeming with islands to explore! It’s a pirate-based adventure game that sees you exploring, trading, fishing, fighting and much more.

Mobile gaming is a big market where the competition is harsh, once in a while you’ll find a game which pushes the limits even further in terms of gameplay, creativity and adventure. Seabeard is one of those games. When we were approached by Handcircus to build the backend we were very excited. We’ve built web services for many applications before but this was our first large scale game.

A game such as Seabeard which involves trading and interacting with the game universe and its other players requires a lot of web services. We built a platform which responds quickly with an optimised code base featuring strong security to avoid fraud, not to mention In App purchases, push notifications, strong data integrity, with a fast and lightweight communication protocol.

At Lost Property we believe in the value of Automated Testing; on Seabeard it was even more crucial to ensure that every aspect of the game was tested and working smoothly. As opposed to a standard web application, identifying an error via a game is much more difficult.

We loved working with Handcircus and being part of something as big as Seabeard with its millions of users. Seabeard is a project that we will be proud of for a long time and we hope to be able to repeat again within the game industry.
